## How to Contribute

- Fork the repository you want to contribute to.
- Create a feature branch (`git checkout -b feature-branch`).
- Commit your changes (`git commit -m 'Add new feature'`).
- Push to the branch (`git push origin feature-branch`).
- Open a Pull Request.
